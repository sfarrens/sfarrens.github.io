Python Set-up for OSX

By Samuel Farrens, Sat 10 December 2016, in category Python

Seting up Python on Mac OSX

Contents

  • Installation
  • Environment
  • Execution
  • Packages

This objective of this post is to help you set up Python 2.7 on Mac OSX. I am currently running 10.12 (Sierra), but most of these suggestions should be applicable to any OS.

I may write another post for help with Python 3 in the future.

Installation

A default version of Python 2.7 is actually pre-installed on OSX.

$ which python
/usr/bin/python

You will find, however, that very few people recommend using the default installation. While the default version certainly works, it can be cumbersome to install and manage packages. So, if you plan to develop in Python, you will probably want to install it yourself.

If you have not already done so you will need to install Xcode Command Line Tools by running:

$ xcode-select --install

There are plenty of ways to install Python yourself, here I will simply focus on 3 easy methods.

1. Anaconda: Anaconda is an open source Python and R distribution platform (and my personal preference). You can download either a graphical installer (.pkg) or command line installer directly from their website. Anaconda comes with a lot of useful pre-installed packages and offers excellent management resources. If, however, you want to keep things as simple (clean) as possible, you may prefer one of the following options.

2. Homebrew: Homebrew is an open source package management system for OSX. To install Homebrew simply run the follow command:

$ /usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"

On the Homebrew website you can find out more about the various packages available. To install Python using Homebrew run the following:

$ brew install python

This will install a Python binary to /usr/local/bin.

3. MacPorts: MacPorts is an alternative package management system for OSX. Details for installing MacPorts are available on their website. To install Python using MacPorts run the following:

$ sudo port install python

This will install a Python binary to /opt/local/bin.

Environment

Now that you have installed Python you can set up your Python evnvironment so that your new Python becomes the default version.

To do this you will need to add the new Python binary to your path by running the following command or adding it to your .bash_profile.

$ export PATH="PATHTOPYTHON/bin:$PATH"

Where PATHTOPYTHON would be /usr/local for a Homebrew installation, /opt/local for a MacPorts installation or something like $HOME/anaconda for an Anaconda installation.

Now you can check which version of Python you are running. e.g. for Hombrew:

$ which python
/usr/local/bin/python

You may also want Python to be able to access the modules you develop yourself. To do this you can set up a PYTHONPATH. For example if you place your Python codes in the directory /Users/username/Documents/Python you can add the following to your .bash_profile.

$ export PYTHONPATH="$HOME/Documents/Python"
$ export PATH="$PYTHONPATH:$PATH"

Execution

To run an interactive Python session simply run:

$ python

This will open the standard Python session. Some nice alternatives to this are iPython and bPython.

To execute a script, you need to save a file with the extension .py. For example, to execute a simple hello world script you can create a file called hello_world.py that contains the following

print 'Hello World!'

and run

$ python hello_world.py
Hello World!

You may wish to make this script an executable and bypass the python command. To do this you need to update your hello_world.py script as follows

#! /usr/bin/env python
print 'Hello World!'

change the file permissions and then run it.

$ chmod +x hello_world.py
$ ./hello_world.py
Hello World!

Packages

Finally, to really take full advantage of what Python has to offer you will want to be able install packages that provide additional functions. The easiest way to do this is with pip. pip comes preinstalled with Anaconda and is usually installed automatically by Homebrew and MacPorts.

To install a package using pip, for example numpy, simply run

$ pip install numpy

then you can import the package in a Python session or script.

import numpy as np

If you want to preload certain packages for all interactove sessions simply create a file called .pythonrc that contains the import commands and save it to $HOME.

If you made it this far you are ready to start developing!
