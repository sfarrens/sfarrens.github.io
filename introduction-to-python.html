Getting Started

A nice way to get a feeling for philosophy behind Python is to run the following command in an interactive Python session (see Python Set-up for OSX).

>>> import this

Documentation

Reading and understand coding languages can be challenging especially for when reading code written by someone else. One of the most important habits one should try to establish early on (and this goes for any language) is adding clear and detailed comments throughout the code. Python is no exception! This practice will improve the readability and facilitate the understanding of the code. Commenting in Python is very easy. Every comment line simply needs to begin with a hash (#).

# This is an example of a comment.

Alternatively comments can be placed between single or double quotes.

''' This is another comment. '''
""" This is also a comment. """

This latter approach is often implemented for documenting files (modules), functions (methods) and classes, but this will be discussed later.

Native Python

Check out this notebook for some an introduction to some of the basic properties of native Python.

Here are a few highlights.

Operators

Native python supports various mathematical operators (e.g. addition, subtraction, multiplication, division, etc.) and logic operators (e.g. and, or, not, etc.). These operators do not require any declaration of variables. In fact, one can easily treat a given Python session like a sophisticated calculator. See here for more details.

For example, to implement the following calculation

\begin{equation*} \frac{9^2}{2} + 2 \end{equation*}

you would do this in Python

>>> 9 ** 2 / 2 + 2
42

You can also define variables to store values for calculations.

>>> x = 11
>>> y = 3
>>> x - y
8

Arrays

The are various ways to store multiple values in a single object. Some of the most basic types are list, tuple and dict.

Lists are mutable (i.e. can be changed)

>>> a = [1, 2]
>>> a[0] = 5
>>> a
[5, 2]

while tuples are immutable (i.e. cannot be changed).

>>> a = (1, 2)
>>> a[0] = 5
Traceback (most recent call last):
  File "<input>", line 1, in <module>
TypeError: 'tuple' object does not support item assignment

Both objects allow concatenation.

>>> a = [1, 2]
>>> b = [3, 4]
>>> a + b
[1, 2, 3, 4]

Dictionaries are used for storing key : value pairs.

>>> a = {'day' : 14, 'month' : 'March', 'year' : 2017}
>>> a['day']
14
>>> a['month']
'March'

Loops

Loops can be implemented in a fairly familiar way (for those who have some coding experience in another language) using a for statement.

>>> a = [1, 2, 3, 4]
>>> for i in range(len(a)):
...     print a[i]
...
...
1
2
3
4

Here len provides the length of the list a and range produces a list of integers from 0 to 3 (i.e. the indices of the list).

A more elegant way of doing this would be to simply iterate through the list. Fortunately Python allows this.

>>> a = [1, 2, 3, 4]
>>> for value in a:
...     print value
...
...
1
2
3
4

Here value is simply a dummy variable. Note: This also works for tuples.

Another elegant option provided in Python is list comprehension, which converts one list into another.

>>> a = [1, 2, 3, 4]
>>> b = [value ** 2 for value in a]
>>> b
[1, 4, 9, 16]

In this example the operation iterates through the elements in the list a, stores each element in a dummy variable value and finally adds the square of the variable to a new list b.

Functions

Functions are an essential part of any programming language and should be used as often as possible to avoid repeating lines of code (which can lead to bugs).

Functions (or methods) are defined using the def statement and results are returned with the return statement. For example, to define a function that adds two values you would do the following

>>> def add(a, b):
...     return a + b
...
>>> add(3, 5)
8

Virtually all Python objects can be passed to functions including other functions. For example, we can pass the add function we just defined to a new function as follows

>>> def operate(a, b, func):
...     return func(a, b)
...
>>> operate(6, 4, add)
10

Classes

Classes can be an extremely powerful tool when used correctly in your code. As a basic rule of thumb you should not create a class unless you actually need a class. Most of the time functions will be sufficient.

Classes are defined using the class statement. A simple example of a class (which need not be a class) is given below

>>> class exampleClass(object):
...
...     def __init__(self, a):
...         self.a = a
...
...     def add(self, b):
...         return self.a + b
...
...     def multiply(self, b):
...         return self.a * b
...
...
>>> x = exampleClass(3)
>>> x.add(7)
10
>>> x.multiply(4)
12

In this example a class called exampleClass is defined. Then a special __init__ method is defined that takes two arguments: the class instance self and an input value a. This method sets the value of the instance variable self.a. The other two methods return operations on the instance variable. x is a class instance with self.a=3.

Numpy

Numpy is a numerical python packing for scientific computing. If you only ever install one Python package this should be it.

To install numpy simply run the following command in a terminal (providing you have already set up pip)

$ pip install numpy

The most essential numpy object is the numpy array (numpy.ndarray).

>>> import numpy as np
>>> a = [1, 2, 3, 4]
>>> type(a)
<type 'list'>
>>> b = np.array(a)
>>> type(b)
<type 'numpy.ndarray'>

In this example the numpy package is imported with the import command. The as assigns the numpy package contents to the object np (otherwise we would have to write numpy.array). The np.array() command converts the list a to a numpy array b.

Numpy arrays have several fantastic properties. Some basic examples are the following

>>> a = np.random.randint(0, 9, 20)
>>> a
array([6, 4, 2, 6, 2, 3, 3, 2, 1, 1, 1, 6, 4, 7, 3, 7, 3, 7, 6, 7])
>>> a.size
20
>>> a.mean()
4.0499999999999998
>>> a.std()
2.1788758569500923
>>> a[2:6]
array([2, 6, 2, 3])

In this example a numpy array of twenty random integers with values between 0 and 9 is created. The size of the array is displayed using the size command. The mean and standard deviation of the array elements are calculated using the mean() and std() commands, respectively.

Numpy also handles linear algebra. For example to take the dot product of two matrices

>>> x = np.random.ranf((3, 3))
>>> x
array([[ 0.6261871 ,  0.71889385,  0.17439468],
       [ 0.19782067,  0.05696277,  0.60367111],
       [ 0.64407532,  0.23033306,  0.45920846]])
>>> y = np.random.ranf((3, 3))
>>> y
array([[ 0.64828158,  0.58614609,  0.05702741],
       [ 0.02461247,  0.61484314,  0.10994705],
       [ 0.98543645,  0.83008403,  0.56223184]])
>>> np.dot(x, y)
array([[ 0.59549419,  0.95380631,  0.21280033],
       [ 0.72452501,  0.65207273,  0.35694721],
       [ 0.87573199,  0.90032254,  0.32023601]])

Matplotlib

To get started with your research you will most likely want to be able to plot your results. After installing numpy, the second package you will want to install is Matplotlib.

To install matplotlib simply run the following command in a terminal

$ pip install matplotlib

To plot \(x^2\) you would do the following

>>> import matplotlib.pyplot as plt
>>> x = np.arange(100)
>>> y = x ** 2
>>> plt.plot(x, y, 'b-')
[<matplotlib.lines.Line2D object at 0x107be4190>]
>>> plt.xlabel('$x$', fontsize=20)
<matplotlib.text.Text object at 0x111977dd0>
>>> plt.ylabel('$x^2$', fontsize=20)
<matplotlib.text.Text object at 0x107bff110>
>>> plt.show()

or to display the contents of a matrix as an image you would do the following

>>> x = np.zeros((11, 11))
>>> x[5] = 1
>>> x[:, 5] = 1
>>> plt.imshow(x)
<matplotlib.image.AxesImage object at 0x107999fd0>
>>> plt.show()
